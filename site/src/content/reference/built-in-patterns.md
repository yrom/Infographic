---
title: 内置图案
---

图案用于风格化类型为 `pattern` 时指定填充样式，使用方式如下：

```json
{
  // 其他配置项...
  "themeConfig": {
    "stylize": {
      "type": "pattern",
      "pattern": "line" // 图案 ID
    }
  }
}
```

AntV Infographic 内置了以下图案，并会在后续持续更新和扩展：

- 线条图案（`line`）：由平行线条组成的图案，适用于表现纹理效果。
- 点状图案（`dot`）：由均匀分布的点组成的图案，适用于表现颗粒感。
- 六边形图案（`hex`)：由六边形网格组成的图案，适用于表现结构化效果。
- 菱形图案（`diamond`）：由菱形网格组成的图案，适用于表现几何美感。
- 马赛克图案（`mosaic`）：由不规则形状组成的图案，适用于表现多样化效果。
- 方形图案（`square`）：由均匀分布的方形组成的图案，适用于表现现代感。
